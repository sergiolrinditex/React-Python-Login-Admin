#### Cambio 2 — `backend/tests/integration/conftest.py` (NUEVO, safety net)

Crear conftest dedicado a `integration/` con autouse de scope `session` que garantiza schema=head al ARRANCAR la suite y restaura head al TERMINAR. No corre por cada test (no degrada perf), pero protege contra futuras regresiones similares.

```python
"""
Hilo People — pytest conftest for backend/tests/integration/.

Slice:  P04-S01-T010 — Test isolation: migrations downgrade pollutes RAG/MCP
Phase:  P04
Purpose: Session-scoped safety net that guarantees the integration suite begins
         and ends with `alembic upgrade head`. Defends against future tests that
         leave the DB in `base` state (the original bug from
         test_migrations_0001_auth.py reset_migrations fixture).

         This fixture is *additive* — individual tests may downgrade locally
         within their own scope as long as their teardown restores head.

Key deps:
  - alembic==1.18.4 (CLI binary)
  - DATABASE_URL env var
"""
from __future__ import annotations
import os, subprocess
from pathlib import Path
import pytest

_BACKEND_DIR = Path(__file__).resolve().parents[2]  # → backend/

def _find_alembic() -> str:
    for c in [Path.home()/"Library"/"Python"/"3.11"/"bin"/"alembic",
              Path("/opt/homebrew/bin/alembic"), Path("/usr/local/bin/alembic")]:
        if c.exists():
            return str(c)
    return "alembic"

@pytest.fixture(scope="session", autouse=True)
def _integration_schema_head():
    """Ensure DB is at alembic head before AND after the integration session."""
    db_url = os.getenv("DATABASE_URL", "postgresql+psycopg://hilo:hilo@localhost:5432/hilo_dev")
    env = {**os.environ, "DATABASE_URL": db_url}
    alembic = _find_alembic()
    subprocess.run([alembic, "upgrade", "head"], cwd=str(_BACKEND_DIR),
                   env=env, capture_output=True, text=True, timeout=120, check=True)
    yield
    subprocess.run([alembic, "upgrade", "head"], cwd=str(_BACKEND_DIR),
                   env=env, capture_output=True, text=True, timeout=120, check=False)
```
