### 2.1 Acceptance expandido (testeable, numerado)

A1. `retrieve(language='es', ...)` devuelve solo chunks cuyos `documents.language='es'`. Chunks de `documents.language='en'` o `'fr'` no aparecen.
A2. `retrieve(collection_ids=[C1], ...)` devuelve solo chunks cuyos `documents.collection_id IN [C1]`. Otras colecciones quedan excluidas.
A3. Combinación AND `language='es' AND collection_ids=[C_es_active]` filtra ambas condiciones simultáneamente.
A4. Colecciones con `rag_collections.enabled=false` se excluyen aunque contengan chunks que cumplen language/collection_id.
A5. Cada resultado expone metadata suficiente para `message_citations`: `chunk_id` (UUID), `document_id` (UUID), `collection_id` (UUID), `score` (float ∈ [0.0, 1.001]; tolerancia por aritmética flotante), `content` (str), `chunk_index` (int), `metadata` (dict; passthrough de `document_chunks.metadata` JSONB), `language` (literal `es|en|fr`).
A6. Parámetro `k` (default=5, rango `[1, 50]` validado por Pydantic) limita el número máximo de resultados devueltos.
A7. `collection_ids` con UUID inexistente devuelve lista vacía (no error).
A8. Búsqueda contra DB vacía (sin chunks ni embeddings) devuelve lista vacía.
A9. `language` fuera de `{es, en, fr}` lanza `pydantic.ValidationError` desde `RetrieverFilters` (Literal).
A10. Embedding con dimensión ≠ 1536 lanza `InvalidEmbeddingDimensionError` (typed domain error) antes de tocar la DB.
A11. Resultados ordenados por similitud descendente (`score` no creciente).
A12. Logging BEFORE/AFTER con `request_id` (ENABLE_VERBOSE_LOGGING=true muestra flujo completo sin contenido de chunks ni floats del embedding; con `false` solo warning+error).

## 6. Front → Back → DB contract

### 6.1 Entrypoint del retriever
Este slice expone una API Python pura (no HTTP). Contrato:

```python
# backend/app/rag/__init__.py — public surface
from app.rag import retrieve, RetrievedChunk, RetrieverFilters
from app.rag import InvalidEmbeddingDimensionError, InvalidLanguageError
```

Firma (use case):

```python
def retrieve(
    *,
    session: Session,                       # SQLAlchemy sync Session, caller-owned
    query_embedding: Sequence[float],       # length must == 1536
    filters: RetrieverFilters,              # Pydantic v2, language/collection_ids/k
    request_id: str | None = None,          # X-Request-ID para correlación de logs
) -> list[RetrievedChunk]:
    """Cosine-similarity search filtered by language + enabled collection.

    Returns at most filters.k chunks ordered by score descending.
    Raises InvalidEmbeddingDimensionError if len(query_embedding) != 1536.
    Raises pydantic.ValidationError via RetrieverFilters if language invalid.
    READ-ONLY: no INSERT/UPDATE/DELETE.
    """
```

### 6.2 Tipos públicos

```python
@dataclass(frozen=True)
class RetrievedChunk:
    chunk_id: uuid.UUID
    document_id: uuid.UUID
    collection_id: uuid.UUID
    language: str          # 'es'|'en'|'fr'
    score: float           # 1.0 - cosine_distance ∈ [0.0, 1.001]
    content: str
    chunk_index: int
    metadata: dict[str, Any]   # passthrough de document_chunks.metadata

class RetrieverFilters(BaseModel):           # Pydantic v2 frozen
    language: Literal["es", "en", "fr"]
    collection_ids: list[uuid.UUID] = []
    k: int = Field(default=5, ge=1, le=50)
```

### 6.3 Cómo lo consume el caller (forward-compat)
- **P02-S03-T002 (chat streaming)** importa `retrieve` y consume `RetrievedChunk` para:
  1. Inyectar contenido en el prompt del modelo (no es responsabilidad del retriever).
  2. Construir filas en `message_citations(id, message_id, document_id, chunk_id, label, score)` después de persistir el mensaje.
  3. `label` lo construye el caller (e.g. "Política Vacaciones ES · p1"), **no** el retriever (D-RET1).
- **P02-S08 (agents)** importará `retrieve` vía `agents/tools/rag_tool.py`. Mismo contrato.
- El retriever **no** registra `llm_usage_logs` (responsabilidad del caller que invocó LiteLLM para embeddings).

### 6.4 SQL contract (referencial — la implementación lo expresa con SQLAlchemy ORM)

```sql
-- distance via pgvector <=> cosine operator (HNSW pushdown)
SELECT
  dc.id              AS chunk_id,
  dc.document_id     AS document_id,
  d.collection_id    AS collection_id,
  d.language         AS language,
  dc.chunk_index     AS chunk_index,
  dc.content         AS content,
  dc.metadata        AS metadata,
  1.0 - (de.embedding <=> :q)  AS score
FROM document_embeddings de
JOIN document_chunks dc ON dc.id = de.chunk_id
JOIN documents d        ON d.id  = dc.document_id
JOIN rag_collections rc ON rc.id = d.collection_id
WHERE d.language = :language
  AND rc.enabled = true
  AND de.embedding IS NOT NULL
  AND (:apply_coll = false OR d.collection_id = ANY(:collection_ids))
ORDER BY de.embedding <=> :q ASC      -- distance ASC == similarity DESC
LIMIT :k;
```
