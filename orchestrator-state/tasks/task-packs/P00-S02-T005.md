- **R6 — RAG docs JSON refs a binaries no existentes**:
  Cada `documents/*.json` referencia un PDF en `binaries/`. Si el helper de
  generación de PDF falla, el checksum_sha256 no se puede computar. Generar
  PDFs PRIMERO, calcular sha256 del bytes resultante, luego inyectar en JSON.
- **R7 — `_bundle_type` propagation**:
  La `BundleType` debe llegar a CADA validator de cada schema. Si solo se
  propaga al loader y los validators ven None, el guard productivo no se
  activa y un developer futuro puede perder la guard. Implementar via
  `model_validator(mode="before")` que lee `cls.__pydantic_validator__.context`
