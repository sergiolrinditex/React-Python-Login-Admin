## Bootstrap note

The `--only auth` bootstrap only seeds the employee_primary user.
The admin user requires `--only admin_ai` (loaded as part of the admin group).
The employee_primary MFA being enabled required using the admin user for direct curl tests.
This is expected behavior per verification data contract design.
