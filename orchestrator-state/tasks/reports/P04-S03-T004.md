   row classification for T004. Ref: task pack §Slice intent.

   and `features/chat/` tracked by P04-S01-T009 (promoted critical FU) are explicitly
   excluded from the 26/26 acceptance gate. Tester invocation uses
   `src/features/admin src/pages/admin/usage` filter. Ref: `HILO_PEOPLE_TECHNICAL_GUIDE.md`,
   handoff tester notes.

3. **i18n count regression classified pre-existing** — `i18n.test.ts toHaveLength(9)` while
   `I18N_NAMESPACES` has 10 entries: regression originated from P04-S02-T005 (commit `5cfcf02`)
   which added "agents" namespace without updating the test. T004 scope (CODE_CHANGES=none)
   cannot touch `i18n/__tests__/i18n.test.ts` (shared file, outside T004 write_set). Will be
   resolved by P04-S03-T001 (bumps count to 11). Not registered as a FU from T004 since
   P04-S03-T001 resolution is imminent (developer done, PROGRESS.md L9).

4. **UsagePage.tsx 316 LoC** — marginally over the 300-line soft cap. T002 validator noted as
   a YAGNI nit; not refactored in T004 because CODE_CHANGES=none is the T004 contract
   (§D-T004-VERIFY-AND-CLOSE in task pack).

5. **TanStack Query v5 `useQuery` as first project use** — ADR-002 (TECHNICAL_GUIDE) records
   the decision to adopt TanStack Query v5 for the UsagePage as the first use-case; staleTime
