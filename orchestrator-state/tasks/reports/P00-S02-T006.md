## Decisions

| Decision | Rationale | Source ref |
|----------|-----------|------------|
| SELECT-then-INSERT diff pattern (vs ON CONFLICT DO NOTHING) | Accurate `added` vs `existing` count in response; idempotency still safe via UNIQUE index at DB level | TECHNICAL_GUIDE §10.3 |
| `del plaintext_key` after client creation in service.py | Minimal cleartext lifetime; Fernet key evicted from local namespace before any other I/O | `01-non-negotiables.md §Security` |
| `_safe_base_url()` strips query string before logging | Defense-in-depth: Gemini passes `key=` as query param; prevents key from appearing in app-level log lines | `01-non-negotiables.md §Security §Logging` |
| `require_admin` P00 stub validates `Bearer dev-admin-` prefix | Placeholder only; explicitly documented for replacement in P01-S02-T001 with RS256 JWT verifier | TECHNICAL_GUIDE §10.2, handoff R3 |
| `fernet_test_key` autouse session-scoped fixture generates real Fernet key | `.env` placeholder `dev-encryption-key-placeholder` is not valid Fernet base64; real key needed for encrypt→decrypt cycle | `01-non-negotiables.md §Tests are REAL` |
| `entity_id` passed as UUID (not str) in audit_log | Matches §10.3 PostgreSQL UUID column type; no silent string coercion | TECHNICAL_GUIDE §10.3 |
| `request_id` placed in `metadata_col` JSONB (not top-level column) | `audit_logs` top-level columns added in P01-S01-T005 (FU-20260509105319); interim placement correct per pack D2 agreement | TECHNICAL_GUIDE §10.3 |
