## Decisions

| Decision | Summary | Source ref |
|---|---|---|
| D-CHATSTREAM-WIRE | Streaming router included in `chat/routers/__init__.py` aggregator — NOT `main.py`. P-22 pattern. | TECHNICAL_GUIDE §5.1 / task pack §H |
| D-CHATSTREAM-SSE (Option B) | `StreamingResponse` with manual SSE byte framing via `sse.py`, not `EventSourceResponse` from `fastapi.sse` (≥0.135.0 not pinned). Wire format identical. | TECHNICAL_GUIDE §3.5.3 / official-doc note P02-S03-T002-fastapi-sse-2026-05-13.md RESOLVED |
| D-CHATSTREAM-PARTIAL | On `asyncio.CancelledError` or LLM mid-stream error: persist partial content with `token_count=None` in a fresh background `_SessionLocal()`. "No silent loss" invariant §12 Constraints. | Task pack §G.2, §K-CHATSTREAM-BGSESSION |
| D-CHATSTREAM-CITORDER | Citations emitted before first chunk (sources shown immediately). Better UX per task pack §H. | Task pack §H |
| D-STREAM1 | `litellm.acompletion(stream=True, stream_options={"include_usage": True})`. Usage on last chunk (`choices=[]`). Guard: `if chunk.usage is not None`. | Official-doc note M.1 CONFIRMED |
| D-COST1 | `litellm.completion_cost(completion_response=chunk)` for `estimated_cost`. Fallback to `ai_models.pricing` JSONB if LiteLLM returns nothing. | Task pack §D-CHATSTREAM-COST |
| D-CANCEL1 | `await _safe_aclose(response)` on `asyncio.CancelledError` or `StopAsyncIteration` to avoid HTTP/2 stream leaks. | Official-doc note M.2 CONFIRMED |
| D-EMBED1 | `litellm.aembedding(model=..., input=[text])` → `result.data[0].embedding`. Dimension enforced client-side. | Official-doc note M.3 CONFIRMED |
| §K-PROMPT-FILE | `SYSTEM_PROMPT_PEOPLE_ASSISTANT` constant inline in `service.py`. YAGNI deferred to versioned file when ≥2 prompts exist. | TECHNICAL_GUIDE §10.4, YAGNI (task pack §G.1) |
| N1 debugger fix | `except (ValidationError, Exception)` on router.py:114 simplified to `except Exception` (ValidationError IS-A Exception in Pydantic v2). Unused import removed. | Validator cycle 1 N1, debugger cycle 1 |
