- `orchestrator-state/tasks/evidence/P01-S02-T010/check-task-dag-strict.log` — shows `Task DAG: INVALID`
- `orchestrator-state/tasks/evidence/P01-S02-T010/test-check-task-dag-strict.log` — shows `Task DAG: INVALID`
- `orchestrator-state/tasks/evidence/P01-S02-T010/verify-slice-rerun.log` — shows `Task DAG: INVALID` and `STRICT_EXIT_WITH_PIPEFAIL=1`

```

WARNING: P00: 21 tasks exceeds max 20; split by screen/module lane
WARNING: P00-S02: 16 tasks exceeds max 15; split the step only if it mixes unrelated screen/API lanes
```
