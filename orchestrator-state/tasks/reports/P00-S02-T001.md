### Known production gap (deferred)

- **Frontend nginx config wiring** (validator §Hallazgo 2, non-blocking now): `infra/nginx/default.conf` is NOT wired into the frontend image (build context is `./frontend`, config lives at `./infra/nginx/`). The compose `frontend:` service also has no bind-mount for it. Consequence when frontend is eventually built: SPA deep links return 404 (no `try_files $uri /index.html`), no `/api/v1` proxy, no security headers. Deferred because `profiles: [frontend]` gates the service from default `up`. Must be resolved before P05/P06 (release/deploy).
