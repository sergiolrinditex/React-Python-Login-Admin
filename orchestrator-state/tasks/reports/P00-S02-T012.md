## Decisions

| Decision | Reference |
|----------|-----------|
| `echo=False` permanent (not `echo="debug"` or runtime-toggled) — KISS over selective redaction via `hide_parameters`. SA echo is a third-party stdlib channel that bypasses the structlog redaction processor; permanent silence is the simplest and safest fix. | HILO_PEOPLE_TECHNICAL_GUIDE §10.5; official-doc-notes/P00-S02-T012-...-structlog-stdlib-2026-05-10.md §5 |
| Pattern coherence with T009: T009 silenced httpx via `setLevel(WARNING)`; T012 silences SA via `echo=False`. Both close third-party stdlib leak vectors at the source. | HILO_PEOPLE_TECHNICAL_GUIDE §10.5 |
| `hide_parameters` (SAv2.0 feature) left as a future reference note in the docstring. It only matters when `echo=True`; since we set `echo=False`, it adds no value now. | Official-doc-note §10.2 / §6 |
| Module docstring extended with CWE-532 chain position + regression guard reference — this is the canonical "no logs modified to remove logs" resolution: SA echo is a third-party channel, not a project structured log. §Security wins (same precedent as T009 httpx suppression). | `.claude/rules/01-non-negotiables.md §Logging` |
