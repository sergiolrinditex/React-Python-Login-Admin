## Important decisions

- `§D-T004-EMAIL-MASK`: maskEmail() is a module-private pure function in ResetSentPage.tsx.
  Algorithm: keep first char of local part + "***" + "@" + domain unchanged.
  Edge cases: empty local (e.g. @example.com) → ***@example.com; missing @ → ***; empty string → ***.
  Justification: tiny (~12 lines), single consumer, no reuse signal yet (YAGNI).

- `§D-T004-NO-STATE-FALLBACK`: when location.state?.email is absent/invalid/non-string,
  renders body.fallback (generic anti-enum copy). No redirect, no error state. Users
  may arrive via deep link or browser refresh after T003 navigated forward.

- `§D-T004-ANTI-ENUM-UI`: both body.with_email and body.fallback use "if registered in Hilo"
  phrasing. The page is oblivious to whether the email was found in DB. Aligns with
  §D-T003-ANTI-ENUM-UI and TECHNICAL_GUIDE §6.2.

- `§D-T004-NO-PII-LOG`: verbose logging emits only email_present(bool) + email_domain(string)
  + email_local_len(number). Never logs raw email, masked email, or any PII.

- `§D-T004-PUBLIC-ROUTE`: mounted OUTSIDE RequireAuth per TECHNICAL_GUIDE §6.4. Authenticated
  users who land here (e.g., testing from /account) see the page normally — no redirect
  to /chat, as that would create a confusing loop.

- `§D-T004-A11Y`: single h1 with title text (Wordmark is wrapped in a div, not h1, to avoid
  dual h1). role="status" + aria-live="polite" on the confirmation region. CTA is a real
  react-router <Link> rendering as <a href="/auth/sign-in"> — keyboard-navigable, min-height
  44px (via inline style matching SolidCTA contract, since SolidCTA doesn't support `as` prop).

- `§D-T004-I18N-LOCKSTEP`: 4 keys in all 3 locales. {{maskedEmail}} ONLY in body.with_email.
  Productive copy confirmed anti-enumeration tone. i18n.test.ts extended with 10 lockstep
  assertions (3 title resolves + 3 body.with_email interpolation + 4-key-per-locale ×3).

- SolidCTA `as` prop not supported: SolidCTA always renders a `<button>`. For the navigation
  CTA, a styled `<Link>` with SolidCTA-matching inline CSS (ink bg, paper fg, tracking-label,
  uppercase, min-height 44px, no border-radius) is used instead. Design-token enforcer passes.

- node_modules symlink: worktree has no local node_modules (Git-ignored symlink target).
