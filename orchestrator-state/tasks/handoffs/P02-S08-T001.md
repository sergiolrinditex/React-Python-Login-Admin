## Endpoints

All three endpoints: `auth=require_admin` (people_admin or super_admin). Schemas in `app/agents/schemas.py`.

### GET /api/v1/admin/ai/agents (line 35ÔÇô71 schemas.py)
- Response 200: `{"data": [AgentOut], "meta": {"request_id": "..."}}`
- `AgentOut`: id, name, description, enabled, config (JSONB), bound_tools (list of BoundToolOut)
- `BoundToolOut` (line 35ÔÇô50): id, name, server_name, enabled, requires_approval, risk_level
- Errors: 401 (invalid/missing JWT), 403 (non-admin)
- DB: SELECT agents + LEFT JOIN mcp_agent_bindings + JOIN mcp_tools (single query); no writes

### PATCH /api/v1/admin/ai/agents/{agent_id}/tools (line 78ÔÇô96 schemas.py)
- Body: `PatchAgentToolsRequest` ÔÇö `{"tool_ids": ["uuid", ...]}`; empty list = unbind all
- Validation: tool_ids missingÔćĺ400 `AGENT_PAYLOAD_INVALID`; agent not foundÔćĺ404; tool not foundÔćĺ400 `AGENT_TOOL_NOT_FOUND`; tool not approved (enabled=False)Ôćĺ400 `AGENT_TOOL_NOT_APPROVED`
- Semantics: set-replace in one transaction (DELETE existing not in new set, INSERT new)
- Response 200: `{"data": AgentOut, "meta": {...}}`
- Audit: `admin.agent.tools.update`, D-S2 independent session
- Errors: 400, 401, 403, 404

### POST /api/v1/agents/runs (line 103ÔÇô143 schemas.py)
- Body: `CreateAgentRunRequest` ÔÇö `{"agent_id": "uuid", "input": "string 1-4000 chars"}`
- Pre-flight: agent exists + enabled=True (else 409 `AGENT_DISABLED`); rate-limit 5 req/min/user
- Execution: INSERT pending Ôćĺ build DeepAgent graph Ôćĺ UPDATE running Ôćĺ invoke_agent Ôćĺ UPDATE done
- Tools with requires_approval=True excluded (V1: log+exclude, don't fail)
- Response 200: `{"data": {"run_id": "uuid", "status": "done"}, "meta": {...}}`
- On error: UPDATE failed, audit, raise 502 `AGENT_RUN_FAILED`
- Audit: `admin.agent.run.start` + `admin.agent.run.complete` (D-S2 pattern)
- Errors: 400, 401, 403, 404, 409, 429 (rate limit), 502
