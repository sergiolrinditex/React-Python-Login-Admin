### Endpoints

1. `POST /api/v1/admin/ai/models/{id}/test`
   - Auth: Bearer JWT + require_admin (people_admin | super_admin)
   - Request: `{"prompt": str (1–4000 chars), "max_tokens": int (1–4096, default 256)}`
   - Response 200: `{"data": {"id": UUID, "model_id": UUID, "output": str, "latency_ms": int, "estimated_cost": float|null, "status": "success", "created_at": iso}, "meta": {"request_id": str}}`
   - Errors: 400/422 AI_MODEL_TEST_INVALID_PAYLOAD (blank/whitespace/too-long prompt), 401 AUTH_SESSION_EXPIRED, 403 AUTH_FORBIDDEN, 404 AI_MODEL_NOT_FOUND, 404 AI_PROVIDER_CREDENTIAL_NOT_FOUND, 429 RATE_LIMITED (ADMIN_AI_TEST 5/min/IP), 502 AI_PROVIDER_TEST_FAILED (LiteLLM timeout or error), 503 Redis fail-closed
   - Side effects: INSERT ai_model_tests + INSERT llm_usage_logs + INSERT audit_logs (action admin.ai.model.test) on success; INSERT failure ai_model_tests + INSERT audit_logs on failure
   - Rate limit: ADMIN_AI_TEST prefix, 5/min/IP, burst=5, window=60s (Redis sliding window)

2. `GET /api/v1/admin/usage?from=&to=&group_by=&model_id=&provider_id=`
   - Auth: Bearer JWT + require_admin
   - Query params: from (ISO 8601, required), to (required), group_by ('model'|'day'|'model_day', default 'model'), model_id (optional UUID), provider_id (optional UUID)
   - Validation: to > from; window ≤ 90 days; group_by in valid set
   - Response 200: `{"data": {"from": iso, "to": iso, "group_by": str, "rows": [...], "totals": {"tokens_in": int, "tokens_out": int, "estimated_cost": float, "invocations": int, "latency_ms_avg": int|null}}, "meta": {"request_id": str}}`
   - Errors: 401, 403, 422 ADMIN_USAGE_INVALID_PAYLOAD (bad from/to/group_by), 422 ADMIN_USAGE_WINDOW_TOO_WIDE (>90 days)
   - Side effects: NONE (read-only)
   - Audit: NONE (consistent with T001 GET endpoints)
   - Rate limit: NONE
